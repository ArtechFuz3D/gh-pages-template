A lightweight GitHub Pages starter with dark/light theme, modular components, and a modern hero.

Use this template

Deploy this template

  1. Click Use this template (opens GitHub)
  2. Create repository from template (give it a name)
  3. In your new repo, go to Settings → Pages
  4. Select branch: main and folder: / (root), Save
  5. Your site will be available at https://<username>.github.io/<repo>/

Tip: enable GitHub Pages' Enforce HTTPS if available.
